=============
MRP Sale Info
=============

This module extends the functionality of sale_mrp and adds related fields to Manufacturing Orders and Work Orders:

    * Sale order
    * Customer
    * Commitment Date

Usage
=====

New fields are displayed in tree and in form views (Extra information tab).
